×

The API provides information on a particular book which includes the book's title, author, year of publication, and ISBN number, as well as the average rating out of 5 and the total number of reviews that the book has received from goodreads.

In order to access the API, visit this site's URL /api/<isbn>, where <isbn> it is ISBN number of the book that you are requesting information on (be sure to enter the ISBN number without the angle brackets <>).

The response is a JSON object. For example, sending a GET request to 127.0.0.1:5000/api/0380795272 would return the following:

{
"author": "Raymond E. Feist",
"average_score": "3.84",
"isbn": "0380795272",
"review_count": 19668,
"title": "Krondor: The Betrayal",
"year": "1998"
}
